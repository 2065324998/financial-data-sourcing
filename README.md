# Overview

An example product that leverages FinTekkers. This code currently works with version 0.0.1 of the broker and ledger. 

The application:

* Downloads details of securities that have been auctioned, and how much the SOMA program bough
* Checks SOMA holdings to generate buy/sell transactions
* Uploads securities and transactions to the ledger
* Queries the ledger service and generates visualizations in Treasury Holdings.ipynb
